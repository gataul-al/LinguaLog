\# LinguaLog - Персональный лингвистический тренажёр



Консольное приложение для изучения иностранных слов.



\## Функции

\- Добавление, редактирование, удаление слов

\- Фильтрация по тегам и дате

\- Режим тренажёра (3 типа упражнений)

\- Отслеживание ошибок (spaced repetition)

\- Экспорт/импорт в CSV и ZIP



\## Запуск

```bash

python main.py
